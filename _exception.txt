_exception.txt

 - 프로그램 실행 전에 발생하는 오류(error)가 아닌 프로그램 실행 중에 발생하는 오류(exception)
 - 발생하는 예외에 대해 처리하는 방법은 조건문(if)이나 try 구문을 이용하면 됨
    단, 일반적으로 if문이 아닌 try 문으로 예외를 처리함
 - try-except 문
    예외처리를 위한 제어문
    문법 :
        try :
            예외가 발생할 수 있는 실행문
        except :
            예외가 발생했을 때 실행할 실행문들 or pass
            pass : 예외가  발생했을 때 실행할 except 안에 있는 명령들을 실행하지 않고 그냥 넘기는 상황에 사용
        else :
            예외가 발생하지 않았을 경우 실행할 실행문들
            else를 사용하는 경우는 예외가 발생할 만한 코드만 try에 넣을 때임
        finally :
            try로 예외처리를 할 때 무조건 실행되는 영역

        - try는 단독으로 사용할 수 없으며 반드시 except나 finally와 함께 사용해야 함
            try + except / try + except + else / try + except + else + finally / try + finally
        - finally 영역은 함수가 종료(return)되거나 루프문에서 break가 걸려도 실행됨
