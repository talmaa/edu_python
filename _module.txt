모듈(module)
 - 여러 변수와 함수를 가지는 집합체로 표준 모듈과 외부 모듈로 나뉨
 - 표준 모듈 : 파이썬에서 제공하는 모듈
 - 외부 모듈 : 기타 개발자들이 만들어서 제공하는 모듈
 - 문법 : import 모듈명 -> 해당 모듈을 사용하겠다는 의미
 - 모듈내의 변수나 함수를 사용하려면 모듈명.함수명()의 방식으로 사용해야 함
    함수 사용시 모듈명을 생략하려면 import 할때 from 절을 이용하면 됨
    from 모듈명 import 사용할 변수나 함수들을 쉼표를 기준으로 나열
    예) from math import ceil, pi, pow, trunc
     - 함수명들을 나열하기 어려우면 *로 모든함수를 지정할 수도 있음
        from math import *
 - 모듈명을 줄여서 사용하고자 하면 as 절을 이용하면 됨
    import 모듈명 as 사용하려는 이름

1. math 모듈
 - 수학관련 기능들을 모아놓은 모듈
 - 주요 함수 및 변수
    ceil(x) : x를 올림하여 정수로 리턴
    floor(x) : x를 내림 하여 정수로 리턴(더 작은 값으로 리턴)
    pi : pi 값으로 원주율
    pow(x, y) : 지수승(거듭제곱) x의 y승을 리턴
    trunc(x) : x를 버림하여 정수로 리턴(0에 가까운 값으로 리턴)
     - 반올림은 round()함수로 작업하나 math 모듈에 들어있는 것이 아님

2. random 모듈
 - 0이상 1미만의 난수를 발생시키는 모듈 
 - 주요 함수
    raondom() : 0이상 1미만의 난수를 리턴
    uniform(x, y) : x이상 y미만의 실수(float)를 리턴
    randrange(x) : 0 이상 x - 1 의 정수를 리턴
    randrange(x, y) : x 이상 y - 1 의 정수를 리턴
    choice(리스트) : 리정된 리스트에서 하나를 랜덤하게 리턴
    sample(리스트, k=숫자) : 리정된 리스트에서 k에 지정된 수만큼 랜덤하게 리턴
    shuffle(리스트) : 리스트 내의 요소들의 순서를 섞어줌

3. datetime 모듈
 - 날짜와 시간에 대한 함수들의 모아놓은 모듈
