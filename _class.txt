클래스(class)
 - 객체(object)로 여러 가지 값(속성) 기능(메소드)을 가진 데이터 타입
    클래스의 값(속성, 변수), 메소드(기능, 함수)를 묶어서 멤버라고 함
 - 클래스를 기반으로 만들어내는 객체를 인스턴스(instance)라고 함
    일반적으로 객체라고 하면 인스턴스를 의미하는 경우가 많음
 - 클래스 생성
    class 클래스명 :
        클래스 내용

 - 클래스 명 : 첫글자를 대문자로 시작하여 다른 자료형의 변수들과 차별
 - 클래스 내용 : 각종 값을 저장할 변수들과 기능을 담당할 함수(메소드)들을 정의
 - 생성자(constructor) : 클래스 내부에 존재하며 해당 클래스를 기반으로 하는 인스턴스를 생성해주는 메소드
    생성자의 이름은 __init__(self[, 매개변수들]) 로 정해져 있음
    생성자에서는 인스턴스 생성과 변수의 초기화를 주로 작업함
    생성자는 인스턴스 생성시 가장 먼저 동작하는 메소드
    self : 자기 자신을 의미하며, 현재 작업중인 인스턴스를 의미하는 값
     - 단, self 라는 이름 자체가 키워드는 아니므로, 다른 이름으로 사용해도 됨
     - 대부분 self 라는 이름을 그대로 사용함

 - 소멸자(destructor)
    현 인스턴스가 메모리에서 소멸될 때 마지막으로 동작하는 일종의 메소드
    소멸자의 이름은 __del__(self[, 매개변수들]) 로 정해져 있음
    생성자에 비해 많이 사용되지는 않는 메소드

 - 메소드 : 클래스가 가지는 함수로 클래스로 할 수 있는 일들을 정의해 놓은 것
    문법 : def 메소드명(self[, 매개변수들]) :
    파이썬에서는 메소드라는 용어보다 멤버 함수나 인스턴스 함수라는 말을 더 많이 사용함
